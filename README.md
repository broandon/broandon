## Featured Project

### 🏆 FitRivals — Competitive Fitness Platform

> *What if your workouts had stakes?*

FitRivals turns daily activity into head-to-head competition. Players challenge each other in 1v1, 2v2, 3v3, or free-for-all formats, with real Apple Health data as the source of truth.

**Under the hood:**
- **HealthKit background delivery** — activity syncs silently even when the app is closed
- **Pre-aggregated Firestore architecture** — player stats computed and stored at write time, not read time. Fast queries at scale.
- **Widget suite** — Heatmap, Battle Status, and Today's Score widgets via WidgetKit
- **Universal Links** — deep linking via inkwellcode.com so challenge invites land exactly where they should
- **Push notification system** — rebuilt from scratch for reliability and meaningful alerts, not noise
- **Cinematic onboarding** — because first impressions are load-bearing

*Currently in alpha. Built solo.*
